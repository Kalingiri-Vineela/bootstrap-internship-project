# Bootstrap 5 Internship Project

## What this project is about
This project was given as an internship task.
The aim was to learn how to build simple web pages using Bootstrap 5.

## Pages in this project
- Home page
- About page
- Contact page

## How I did this project
First, I created three HTML files.
Then I added Bootstrap using CDN.
I created a menu to move between pages.
I added content and a contact form.
Finally, I tested the pages in the browser.

## Tools I used
- HTML
- Bootstrap 5
- GitHub
- ChatGPT for learning help

## Challenges I faced
I was new to Bootstrap and web development.
Initially, I did not understand how to start.

## How I solved them
I learned step by step through youtube and chatgpt and practiced slowly.
