**Internship Project – Bootstrap 5 UI Design**

**Project Overview**

This project is developed as part of an internship task to explore Bootstrap 5 and build responsive web pages by remixing Bootstrap components. The website represents a simple Campus Placement Assistance platform for students.

**Technologies Used**

HTML5

CSS3

Bootstrap 5

Git \& GitHub

**Pages Included**

Home Page – Navbar, hero section, feature cards, footer

Services Page – List of placement-related services

Contact Page – Contact form layout

**What I Learned**

How to use Bootstrap components effectively

Creating responsive layouts using the grid system

Maintaining consistent design across multiple pages

**Challenges \& Solution**

I faced issues with layout alignment and responsiveness, which I solved by referring to Bootstrap documentation and using utility classes properly.

**External Help**

I used Bootstrap official documentation and AI tools (like ChatGPT) for learning and guidance.
